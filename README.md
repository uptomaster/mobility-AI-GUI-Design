## 💡 Project Concept: No More Pause
우리는 여행의 모든 과정에서 발생하는 **'멈춤(Pause)'**에 주목했습니다. 단순히 물리적인 장애물을 넘어, 여행을 주저하게 만드는 모든 마찰력을 기술로 해결합니다.

* **심리적 멈춤:** "짐을 얼마나 싸야 하지? 어떤 가방이 적당할까?"라는 준비 단계의 고민.
* **물리적 멈춤:** 계단, 소음, 노면의 진동, 그리고 쉴 곳 없는 길 위에서의 피로.

## 🛠 Key Features & Technologies
**오리가미 구조와 강성 제어 기술을 통한 이동의 재정의**

### 1. 상황 대응형 가변 구조 (Origami Structure)
- **부피 최적화:** 짐의 양에 맞춰 스스로 형태를 조절하여 심리적·물리적 무게를 최소화합니다.
- **모드 전환:** 상황에 따라 '메는 가방'에서 '끄는 모듈'로 완벽하게 변형됩니다.

### 2. 바퀴 가변 및 강성 제어 (Stiffness Control)
- **바퀴 유무 결정:** 지형에 따라 바퀴를 숨기거나 전개하여 이동의 효율을 극대화합니다.
- **소프트 휠 기술:** 노면의 소음과 진동을 흡수하여 여행지의 풍경에만 집중할 수 있게 합니다.
- **즉각적 휴식:** 필요한 순간 외장의 강성을 높여 어디서든 단단하고 편안한 나만의 의자가 됩니다.

## 🎨 Design Principle
- **Minimal & Professional:** 불필요한 노이즈를 제거한 'Toss-style'의 정갈한 인터페이스.
- **Adaptive Interaction:** 모듈의 물리적 변화(강성, 부피, 바퀴 상태)를 직관적으로 시각화하는 GUI.
